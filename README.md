# Installation

* `cd ~/code/laptop` or wherever you want to clone this repo.
* `git clone git@github.com:neerajsingh0101/dotfiles.git`
* `cd dotfiles`
* Execute `./install.sh`.
* Execute `./macos/set-defaults.sh`.
